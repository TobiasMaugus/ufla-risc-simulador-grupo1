# Simulador Funcional do Processador UFLA-RISC
Este projeto implementa um simulador funcional para o processador RISC de 32 bits
UFLA-RISC.

## Como Executar
1. Clone o repositório:
git clone https://github.com/TobiasMaugus/ufla-risc-simulador-grupo1.git

2. Compile o simulador:
cd src/simulador
python main.py

3. Execute com um arquivo binário:
python main.py binarios/programa.bin

## Testes
python -m unittest discover
